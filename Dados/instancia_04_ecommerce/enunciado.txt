ATIVIDADE - TESTES DE HIPÓTESE (ECOMMERCE)

Dataset: dataset.csv

REGRAS

- Considere α = 0.05 em todos os itens.
- Arredonde todas as respostas numéricas para 2 casas decimais.
- Cada item gera um número final.
- O seu resultado individual é a soma dos números dos itens.
- O grupo vencedor é o que entregar primeiro a soma correta dos resultados individuais.

ITENS

a) A taxa de conversão mudou depois da intervenção? Compare antes e depois para os mesmos clientes. Use Wilcoxon pareado bilateral com α = 5%. Se houver diferença significativa, a resposta é a maior mediana de conversão entre antes e depois. Se não houver diferença, use a mediana geral entre as duas colunas.

b) Qual campanha parece produzir maior satisfação? Use Mann-Whitney U bilateral com α = 5%. Se houver diferença significativa, a resposta é a maior mediana de satisfação. Se não houver diferença, use a mediana geral da satisfação.

c) Antes de qualquer intervenção, o tempo no site era maior em mobile ou desktop? Use o teste t de Welch bilateral com α = 5%. Se houver diferença significativa, a resposta é a maior média de tempo. Se não houver diferença, use a média geral do tempo antes.

d) Qual campanha gera maior ticket médio, A ou B? Use teste t independente bilateral com α = 5%. Se houver diferença significativa, a resposta é o maior ticket médio entre as campanhas. Se não houver diferença, use o ticket médio geral.
