ATIVIDADE - TESTES DE HIPÓTESE (ECOMMERCE)

Dataset: dataset.csv

REGRAS

- Considere α = 0.05 em todos os itens.
- Arredonde todas as respostas numéricas para 2 casas decimais.
- Cada item gera um número final.
- O seu resultado individual é a soma dos números dos itens.
- O grupo vencedor é o que entregar primeiro a soma correta dos resultados individuais.

ITENS

a) Qual campanha parece produzir maior satisfação? Use Mann-Whitney U bilateral com α = 5%. Se houver diferença significativa, a resposta é a maior mediana de satisfação. Se não houver diferença, use a mediana geral da satisfação.

b) A taxa de conversão mudou depois da intervenção? Compare antes e depois para os mesmos clientes. Use Wilcoxon pareado bilateral com α = 5%. Se houver diferença significativa, a resposta é a maior mediana de conversão entre antes e depois. Se não houver diferença, use a mediana geral entre as duas colunas.

c) Antes de qualquer intervenção, o tempo no site era maior em mobile ou desktop? Use o teste t de Welch bilateral com α = 5%. Se houver diferença significativa, a resposta é a maior média de tempo. Se não houver diferença, use a média geral do tempo antes.

d) Após a mudança da jornada, o tempo médio no site mudou? Compare tempo antes e depois para os mesmos clientes. Use teste t pareado bilateral com α = 5%. Se houver diferença significativa, a resposta é a menor média entre antes e depois. Se não houver diferença, use a média geral entre as duas colunas.
